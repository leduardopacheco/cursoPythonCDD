# Python para Ciencia de Dados: do zero ate EDA

Repositorio com o material didatico de um curso introdutorio de Python aplicado a Ciencia de Dados, desenvolvido como atividade extensionista da disciplina de Ciencia de Dados.

## Objetivo
Oferecer formacao basica em programacao, manipulacao, visualizacao e analise exploratoria de dados para iniciantes.

## Publico-alvo
- Estudantes do ensino medio, tecnico ou superior
- Profissionais em transicao de carreira
- Iniciantes interessados em Python e Ciencia de Dados

## Estrutura do repositorio
- `planejamento/`: documento da Entrega 1
- `slides/`: apresentacoes por modulo
- `notebooks/`: notebooks comentados e funcionais
- `exercicios/`: listas de exercicios e gabaritos
- `datasets/`: conjuntos de dados usados nas atividades
- `materiais-complementares/`: links e referencias

## Conteudo do curso
### Modulo 1 - Fundamentos de Python
- Sintaxe basica
- Variaveis e tipos de dados
- Condicionais e repeticao
- Estruturas de dados
- Boas praticas

### Modulo 2 - Manipulacao de Dados
- Introducao ao Pandas
- Leitura e tratamento de dados
- Transformacoes basicas
- Estatistica descritiva

### Modulo 3 - Visualizacao de Dados
- Fundamentos de visualizacao
- Graficos com Matplotlib e Seaborn
- Comunicacao visual

### Modulo 4 - EDA
- Perguntas norteadoras
- Exploracao de dados
- Geracao de insights
- Comunicacao dos resultados

## Como abrir os arquivos
### Slides
Abra o arquivo `.pptx` no PowerPoint, LibreOffice Impress ou Google Slides.

### Notebook
Use Google Colab ou Jupyter Notebook.

### Requisitos para executar localmente
```bash
pip install pandas matplotlib seaborn notebook
```
